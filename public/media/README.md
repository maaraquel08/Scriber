# Media Files Directory

Place your audio/video files here for testing the transcript editor.

## File Naming

Name your media file using one of these patterns:

1. **By Transcript ID** (recommended):

    - `ee7f8d5afabc0e529dcc9be8c351947c7026f57cbca16217eb81aec8476184c4.mp4`
    - Matches the transcript JSON file ID

2. **Simple name** (fallback):
    - `test-media.mp4`
    - `test-media.mp3`

## Supported Formats

### Video:

-   `.mp4` (recommended)
-   `.mov`
-   `.webm`
-   `.mkv`
-   `.avi`

### Audio:

-   `.mp3`
-   `.wav`
-   `.m4a`
-   `.ogg`

## Example

If your transcript ID is `ee7f8d5afabc0e529dcc9be8c351947c7026f57cbca16217eb81aec8476184c4`, place your file as:

```
public/media/ee7f8d5afabc0e529dcc9be8c351947c7026f57cbca16217eb81aec8476184c4.mp4
```

The app will automatically load this file when the transcript loads.
